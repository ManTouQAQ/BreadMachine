### 特性
***
* 自动注册Listener和Command
* IOC容器支持(基于Guice)
* 基于注解的Bot指令开发
* 更加方便的CardButtonClick事件监听
* 一些基于IOC的特性如自动注入配置文件的obj，init方法
* etc...
